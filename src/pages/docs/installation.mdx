# Installation

Install the package and its peer deps:

```bash
npm install @cladd-ui/react
```

Then import the stylesheet from your app entry (alongside your Tailwind setup):

```css
@import "tailwindcss";
@import "@cladd-ui/react/css";
```

Wrap your app in `UIProvider`:

```tsx
import { UIProvider } from '@cladd-ui/react';

export default function App({ children }) {
  return <UIProvider>{children}</UIProvider>;
}
```
