# Introduction

**cladd** is a React UI primitive library. It ships unstyled-but-not-quite
components that already understand surfaces, focus rings, and theming, so you
can compose them into your own design system without rebuilding the boring
parts.

## What you get

- A small set of building blocks: `Button`, `Surface`, `Dialog`, `Popover`, `List`, ...
- Built on Tailwind v4 and CSS variables — themable from the outside.
- Hooks for the common imperative bits: `useDialog`, `useToast`, `useTheme`.

Head over to [Installation](/docs/installation/) to get set up.
