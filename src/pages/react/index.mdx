# Introduction

**Cladd** is an opinionated React UI kit for building applications. It's not a
collection of headless primitives — it ships with a defined visual identity:
recessed/cut surfaces, subtle gradients, dark-first theming, a lime accent, and
a coherent set of controls that look like they belong in the same product.

If you've used Radix or Base UI, those give you behaviour and leave the design
to you. Cladd gives you both — and a kit complete enough that you can start
shipping real screens immediately.

## What's in the box

- **A surface system.** `Surface`, `SurfaceCut`, and five depth levels that
  nest contextually. Surfaces inherit and offset from their parent, so nested
  panels read correctly without manual color picking.
- **A sizing scale.** Every interactive control — `Button`, `Input`,
  `NumberField`, `OTPField`, `Switch`, `Slider`, `Chip`, `Shortcut` — supports
  the same `2xs → 2xl` size axis. They line up.
- **Eleven accent colors.** `neutral`, `brand`, `red`, `pink`, `purple`,
  `blue`, `cyan`, `lime`, `green`, `yellow`, `orange` — each with the right
  text, fill, and outline shades for any surface variant.
- **Application-grade components.** `Toolbar`, `Segmented`, `Popover`,
  `Dialog`, `Toast`, `Tooltip`, `Shortcut`, `Select`, `OTPField`, `Slider`,
  `NumberField` — the parts you need for the boring shell of a real app.
- **Hooks for the imperative bits.** `useDialog`, `useToast`, `useTheme`,
  `useSurface`, `useDevice`.

## Already shipping

Cladd powers production apps:

- [Swiper Studio](https://studio.swiperjs.com/) — visual editor for Swiper sliders
- [t0ggles](https://t0ggles.com/) — project management
- [PaneFlow](https://paneflow.com) — created stunning slideshows in minutes
- [Start Page HQ](https://startpagehq.com/) — custom New Tab dashboard with 50+ widgets

what a button looks like — start with [Installation](/react/installation/).
